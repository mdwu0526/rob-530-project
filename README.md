# rob-530-project
ROB530 - Mobile Robotics Final Project

Report video: https://www.youtube.com/watch?v=al2p-oy7msg

Dependencies (with tested versions):
==
  Post-processing:
    Python (3.8)
    OpenCV (4.9)
    Scipy (1.10)
    Matplotlib (3.1.2)
    Numpy (1.24)

  For Apriltags:
    dt-apriltags (>=3.1)

How to run the code:
1. Pull the repository from Github/Gitlab
2. Run run.py script
$ python3 run.py
